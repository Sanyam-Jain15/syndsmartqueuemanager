# Instructions To Run

## Web App:
Visit: https://syndsmart.herokuapp.com
or
Install Nodejs
CD into Web Application Directory
Run: npm install in terminal/cmd
wait for a min
visit: 127.0.0.1:3000

## Android APP:
Run APK or
Install Android Studio
Open Android Application Directory
Run via android studio
